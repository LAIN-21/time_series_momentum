## Time-Series Momentum

This repository implements and explores the **Time-Series Momentum** research paper in a reproducible Python environment.

There will be two main notebooks:
- **`introduction.ipynb`**: Introduces the theory behind time-series momentum, the data used, and key concepts from the paper.
- **`application.ipynb`** (planned): Focuses on applying the theory to build, backtest, and evaluate a practical time-series momentum trading strategy.

To get started, install the dependencies:

```bash
pip install -r requirements.txt
```
